# Interview Description

Congratulations on proceeding to the second-round interview! This interview will be one hour long with a panel of roughly 5 members from our team. The agenda is as follows:

 

Introductions ~ 10 min
Short presentation based on evaluation (details below) ~ 20 min
Resume and other technical questions ~ 15 min
Time for questions from the candidate ~ 15 min
 

Please find the materials for the evaluation on GitHub at: https://github.com/archimedes06/Interviews.git.

Since this is the first time you’ve seen this data, please feel free to reach out to me at steven.weinell@spectrum.com with any questions you have.

 

The evaluation has two parts.

 

Sample Data analysis and modeling
In the repo the sampleData.zip folder contains two csv files with data very similar (in type and quality) to what we work with daily.  It also has a readme file with information on the data files.

 

The password for sampleData.zip is: sp3ctrumDataScience!

 

Analyze the data in the csv files and build a model to predict whether a customer will churn (discontinue their Charter service). Expect to spend about two hours on the analysis and modeling.

 

Short presentation (10-15 min)
Please create a one or two slide presentation to explain the results of your modeling. Be sure to include a description of any use of AI for development of your solution; you are welcome add a third slide for this description if needed. There is no need to send us the presentation ahead of time, we will go over it with you at the interview. Also, don’t worry if there are things you would have liked to do, but didn’t have time. We can discuss any further analysis you would like to have done during the meeting.
